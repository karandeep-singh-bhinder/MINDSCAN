      About MindScan
      An ML-driven mental wellness tool built for students, by students.

      MindScan uses a logistic regression model trained on the Perceived Stress Scale (PSS-10) extended with academic, social, and physical well-being features. The model classifies student stress into three levels — Low, Moderate, and High — and provides personalised YouTube resources to support mental wellness. Built as a final-year ML project.

      Technologies Used

      Python scikit-learn Pandas NumPy Logistic Regression PSS-10 Dataset K-Fold CV HTML/CSS/JS

      📊 Dataset Information

      The model is trained on an extended student stress dataset combining the Perceived Stress Scale (PSS-10) with additional features covering academic pressure, sleep quality, social interaction, physical symptoms, and emotional regulation. Dataset size: 980 records, 15 features, 3-class labels. Preprocessing includes Min-Max normalisation, class-weight balancing, and stratified train/test splits (80/20).
